
## plugin-unicodecss

To support unicode font family and font embed in gitbook web version.


### To Install

```
npm install plugin-unicodecss
```

In `book.json` ,

add `plugin-unicodecss` in `plugins`

```
"plugins": [
      "plugin-unicodecss"
      ]
```
