
## gitbook-plugin-notocjk

To support Noto CJK font family and font embed in gitbook web version.


### To Install

```
npm install gitbook-plugin-notocjk
```

In `book.json` ,

add `gitbook-plugin-notocjk` in `plugins`

```
"plugins": [
      "gitbook-plugin-notocjk"
      ]
```
